# Exemples de code corrects

Ce dossier contient des exemples de code **corrects** illustrant les bonnes
pratiques et les concepts abordés dans le cours.

Tous les fichiers dans ce dossier :

- Sont complets et exécutables.
- Suivent les bonnes pratiques de programmation.
- Contiennent des commentaires expliquant les étapes importantes.
- Peuvent être utilisés comme référence pour vos propres projets.

## Comment utiliser ces exemples

1. **Lisez le code** attentivement en suivant les commentaires.
2. **Exécutez-le** sur votre machine pour observer le résultat.
3. **Expérimentez** en modifiant certaines parties pour tester votre
   compréhension.
4. **Utilisez-les comme base** pour réaliser les exercices et projets.

> [!TIP]
>
> Essayez de comprendre le raisonnement derrière chaque ligne de code plutôt que
> de simplement copier-coller. Posez-vous la question : "Pourquoi ce code
> fonctionne-t-il ?"
