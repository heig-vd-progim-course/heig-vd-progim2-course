# Exemple 01 - Déclaration de variables booléennes

**Objectif pédagogique :** comprendre comment déclarer et initialiser des
variables booléennes en Java.

**Concepts illustrés :**

- Type primitif `boolean`
- Valeurs `true` et `false`
- Convention de nommage pour les variables booléennes

## Description

Cet exemple montre comment déclarer et initialiser correctement des variables
booléennes en Java. Le type `boolean` est un type primitif qui ne peut prendre
que deux valeurs : `true` (vrai) ou `false` (faux).

Les conventions de nommage recommandent d'utiliser des préfixes comme `is`,
`has`, `can`, `should` pour rendre le code plus lisible et expressif.

## Code

Le fichier `Example.java` contient des exemples de déclarations de variables
booléennes avec différentes conventions de nommage.

## Sortie attendue

```
État de connexion
isConnected: true
hasError: false
isValidInput: true

Vérifications
canEdit: true
shouldSave: false
isReady: true
```

## Points clés à retenir

- Le type `boolean` est un type primitif en Java
- Une variable booléenne ne peut être que `true` ou `false`
- Utiliser des noms explicites avec préfixes (`is`, `has`, `can`, `should`)
- Les noms de variables doivent indiquer clairement ce qu'elles représentent

## Suggestions d'expérimentation

Pour mieux comprendre cet exemple, essayez de :

1. Ajouter d'autres variables booléennes avec des noms significatifs.
2. Modifier les valeurs et observer comment elles s'affichent.
3. Créer des variables avec des noms peu clairs (comme `flag`, `x`) et comparer
   avec les noms explicites.
